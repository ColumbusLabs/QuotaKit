CodexBar 0.19.0-Mobile 1.1.0

本リリースではアップストリームの CodexBar 0.19.0 をフォークに統合し、完全な互換性を確保するとともに、今後のモバイル機能開発の基盤を整えました。アップストリームの新機能に対応するモバイル版は開発中です。

Mobile 1.1.0（変更なし）

  • iCloud 同期を KVS から CloudKit にアップグレードし、マルチデバイス同期に対応。
  • セッションクォータのプッシュ通知：iOS がバックグラウンドで枯渇/回復を通知。
  • Sparkle 複合ビルド番号でアップストリームとの競合を回避。

CodexBar 0.19.0（アップストリーム）

  • Alibaba Coding Plan プロバイダー（リージョン対応のクォータ取得）を追加。
  • メニューバーにサブスクリプション利用率履歴チャートを追加。
  • Claude プロバイダーのエンドツーエンドリファクタリング。
  • Cursor の使用量をダッシュボードの Total/Auto/API に整合。
  • Codex コードレビュー制限のリセット時間を表示。
  • コスト履歴にモデル別トークン数を追加。
  • GPT-5.4 mini および nano の料金対応。
  • Antigravity のモデル選択を修正。
