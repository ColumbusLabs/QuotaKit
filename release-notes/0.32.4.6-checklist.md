# QuotaKit Mac 0.32.4.6 Release Checklist Notes

## CloudKit Schema Gate

- `Scripts/verify-cloudkit-schema.sh` could not export the Production schema because this Mac does not have a saved CloudKit Management Token.
- This release is being shipped with `QUOTAKIT_SKIP_CLOUDKIT_SCHEMA_VERIFY=1` to deliver the corrected customer Sparkle appcast feed and the already-merged Mac fixes.
- Code review of the release diff from `v0.32.4.5` found no CloudKit schema-bearing record type, query index, or subscription changes.
- The release diff adds optional JSON payload data for synced rate-window identity inside existing encoded snapshot blobs; that does not require a CloudKit Production schema deploy.
- CloudKit container remains `iCloud.com.columbuslabs.quotakit`.
- The skipped verification should be restored for the next non-urgent Mac release after saving a management token with `xcrun cktool save-token`.
