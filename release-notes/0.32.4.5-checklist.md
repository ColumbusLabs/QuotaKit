# QuotaKit Mac 0.32.4.5 Release Checklist Notes

## CloudKit Schema Gate

- `Scripts/verify-cloudkit-schema.sh` could not export the Production schema because this Mac did not have a saved CloudKit Management Token.
- This release is being shipped with `QUOTAKIT_SKIP_CLOUDKIT_SCHEMA_VERIFY=1` because it includes a provider redirect security fix and should not wait on local token setup.
- Code review of the release diff found no CloudKit schema-bearing record type or field changes. The only sync-related changes are provider classification updates for the new Devin provider in existing sync code paths.
- CloudKit container remains `iCloud.com.columbuslabs.quotakit`.
- The skipped verification should be restored for the next non-urgent Mac release after saving a management token with `xcrun cktool save-token`.
